# Лабораторная работа 3: Геометрические фигуры

## Описание
Данный проект реализует иерархию классов для работы с геометрическими фигурами с полным интерактивным меню пользователя, возможностью сохранения и загрузки данных из файлов.

## Функциональность

### Интерактивное меню
Программа предоставляет полноценное интерактивное меню с следующими возможностями:

1. **Добавление фигур** - создание новых фигур с вводом параметров
2. **Просмотр фигур** - отображение всех фигур или конкретной фигуры
3. **Удаление фигур** - удаление выбранной фигуры или всех фигур
4. **Сохранение в файл** - сохранение всех фигур в текстовый файл
5. **Загрузка из файла** - загрузка фигур из текстового файла
6. **Сохранение результатов** - экспорт результатов вычислений
7. **Статистика** - подсчет общих и средних значений
8. **Безопасный ввод** - проверка корректности вводимых данных

### Классы

#### Базовый класс `Figure`
Абстрактный базовый класс с полным набором конструкторов и виртуальных функций:

**Конструкторы:**
- Конструктор по умолчанию
- Конструктор копирования
- Оператор присваивания
- Виртуальный деструктор

**Виртуальные функции:**
- `double getArea() const` - вычисление площади
- `double getPerimeter() const` - вычисление периметра
- `void printInfo() const` - вывод информации на консоль
- `void saveToFile(std::ostream& out) const` - сохранение в файл
- `void loadFromFile(std::istream& in)` - загрузка из файла
- `std::string getType() const` - получение типа фигуры

### Производные классы

#### 1. `Circle` (Круг)
- **Конструкторы:** по умолчанию (r=1.0), с параметром, копирования
- **Параметры:** радиус
- **Формулы:**
  - Площадь: π × r²
  - Периметр: 2 × π × r

#### 2. `Rectangle` (Прямоугольник)
- **Конструкторы:** по умолчанию (1×1), с параметрами, копирования
- **Параметры:** ширина, высота
- **Формулы:**
  - Площадь: ширина × высота
  - Периметр: 2 × (ширина + высота)

#### 3. `Trapezoid` (Трапеция)
- **Конструкторы:** по умолчанию, с параметрами, копирования
- **Параметры:** верхнее основание, нижнее основание, высота, боковые стороны
- **Формулы:**
  - Площадь: (основание1 + основание2) × высота / 2
  - Периметр: основание1 + основание2 + сторона1 + сторона2

#### 4. `Triangle` (Треугольник)
- **Конструкторы:** по умолчанию (3-4-5), с параметрами, копирования
- **Параметры:** три стороны
- **Формулы:**
  - Площадь: по формуле Герона
  - Периметр: сумма всех сторон
- **Дополнительно:** проверка корректности треугольника

## Формулы

### Круг
- Площадь: S = πr²
- Длина окружности: P = 2πr

### Прямоугольник
- Площадь: S = a × b
- Периметр: P = 2(a + b)

### Трапеция
- Площадь: S = (a + b) × h / 2
- Периметр: P = a + b + c + d

### Треугольник
- Площадь (формула Герона): S = √(p(p-a)(p-b)(p-c)), где p = (a+b+c)/2
- Периметр: P = a + b + c
- Условие существования: a + b > c, a + c > b, b + c > a
